# [ 프로젝트의 주요 내용 ]

### :dart: 목표 : Spring MVC 패턴을 이용한 계층형 게시판 구현

#### :pencil: 사용자가 게시물을 작성할 수 있고 게시물에 대한 답글을 달 수 있는 게시판 만들기

#### :pencil: KakaoPay API 사용한 간편 단건 결제 구현하기

#### :pencil: 관리자 프로젝트를 구현해 사용자 데이터 리스트를 게시판 형태로 볼 수 있도록 하고 회원들의 정보를 수정할 수 있도록 하기
